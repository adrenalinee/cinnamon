# Quick start

`$ git clone https://github.com/adrenalinee/cinnamon.git`

`$ cd cinnamon`

`$ mvn install`

`$ cd cinnamin-core-stater`

`$ mvn spring-boot:run`

웹 브라우저에서 http://localhost:8080 접속
